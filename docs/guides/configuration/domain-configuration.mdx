## Steps to Configure Your Domain

### 1. Log Into the Plesk Panel

- **Access Plesk**:
  - Use your **username** and **password** to log into the Plesk Panel.

### 2. Select the Domain

- From the **domain list**, select the domain you wish to configure.

### 3. Configure DNS Settings

- Navigate to the **DNS Settings** option within the selected domain's settings.
- Click on the **Add Record** button.

### 4. Set Up the Record

- **Domain Name Field**:
  - Enter the **subdomain** (e.g., `subdomain.example.com`) or leave it blank for the main domain.
- **IP Address Field**:
  - Enter the **IP address** where the domain should point.
- Click **OK** to save the record.

### 5. Apply Updates

- Confirm the changes by clicking the **Update** button.
- Wait for the changes to propagate (may take a few minutes to 24 hours depending on DNS settings).

This process successfully configures DNS records for your domain.
