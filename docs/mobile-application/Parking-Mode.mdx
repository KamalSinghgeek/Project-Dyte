# Parking Mode

---

The **Parking Mode** feature is designed to monitor unauthorized vehicle movements while the vehicle is parked.

---

### **Parking Mode**:

- **Enable notifications** for any unauthorized movements detected while the vehicle is parked. 
- Stay informed if the vehicle is being tampered with or moved without authorization.
