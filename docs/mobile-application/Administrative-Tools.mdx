# Administrative Tools

---

The **Administrative Tools** section is designed for managing users and vehicles. This section provides administrators with the ability to control who has access to the system and make necessary changes to vehicle information. These tools ensure smooth operations, enhance fleet management, and help maintain system integrity.

---

### **Manage Users**:

This section allows administrators to view and manage users. You can see a list of all active users, update their details, and ensure that each user has the appropriate permissions to access and manage the system.

- **View user lists**: Browse through the list of all users to see who is currently in the system. The list will display user details such as their name and profile name, making it easy to identify each user.
- **Edit user details**: Update user information such as names, roles, contact details, and access rights when necessary. Administrators can modify permissions or adjust user roles to ensure the system is used properly.
- **Login as a user**: Administrators can log into any user profile to troubleshoot or perform tasks from the perspective of that user.

This feature helps administrators manage access efficiently, ensuring that only authorized individuals can access critical information.

### **Add New Users**:

Administrators can add new users by clicking the **Add New User** button. This functionality is particularly useful when new team members need access to the Fleet Stack system. You can assign roles, define access levels, and ensure that only authorized personnel are allowed to manage fleet data.

- **Add new users**: Click the **Add New User** button to create new user profiles. Fill in the necessary details such as name, email, role, and permissions to grant access to the Fleet Stack system.
- **Assign roles and permissions**: Administrators can customize the roles of new users, specifying which features and data they can access. This ensures proper management and security across the system.

This tool enables seamless onboarding of new users and ensures that user permissions are correctly configured from the start.

### **Manage Vehicles**:

The Manage Vehicles tool allows administrators to update vehicle information, view vehicle locations, and send commands to devices installed in the vehicles. This functionality is crucial for maintaining an up-to-date record of all fleet vehicles and their statuses.

- **Edit vehicle details**: Modify vehicle information, including make, model, registration details, and other relevant data. Keeping vehicle records up to date ensures accurate tracking and reporting.
- **Navigate to the vehicleâ€™s location**: View the real-time location of each vehicle on the map. Administrators can zoom in or out on the map to track the fleet's movements and ensure vehicles are on their assigned routes.
- **Send commands to devices**: Administrators can send specific commands to devices within the vehicle, such as adjusting APN settings or modifying operator configurations. This feature allows for remote management of vehicle systems.

This feature provides the tools needed to effectively manage fleet vehicles, ensuring that all information is accurate and accessible when needed.
