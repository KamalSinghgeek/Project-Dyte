# Administrative Tools

---

The **Administrative Tools** section is designed for managing users and vehicles. Here, administrators can control who has access to the system and make changes to vehicle information, ensuring smooth operations and optimal fleet management.

---

### **Manage Users**:

This section allows administrators to view and manage users. You can easily see a list of all users, edit their details, and ensure that everyone has the correct permissions to access the system.

- **View user lists**: Browse through the list of all users to see who is currently in the system.
- **Edit user details**: Update user information such as names, roles, or contact details when necessary.

### **Add New Users**:

You can easily add new users by clicking the **Add New User** button. This is useful when new team members need access to the Fleet Stack system. It ensures that only authorized personnel can access and manage fleet data.

- **Add new users**: Simply click the button to create new user profiles, assign roles, and define access levels.

### **Manage Vehicles**:

This tool allows administrators to manage vehicle information directly from the system. You can update vehicle details, view the vehicle’s current location, and send commands to devices within the vehicle.

- **Edit vehicle details**: Modify information like vehicle make, model, or registration details.
- **Navigate to the vehicle’s location**: See where the vehicle is located in real-time on the map.
- **Send commands to devices**: Send commands to devices installed in the vehicle, such as adjusting APN or operator settings.
