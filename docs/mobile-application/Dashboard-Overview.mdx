# Dashboard Overview
---
- Displays **Total Vehicles** and their status:
  - Running
  - Stopped
  - Idle
  - Inactive
  - No Data
- View **total kilometers** traveled, displayed in a graphical format.
- Quick actions are available for real-time tracking.
