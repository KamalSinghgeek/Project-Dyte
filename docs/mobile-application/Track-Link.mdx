# Track Link

---

The **Track Link** feature allows you to create a shareable link for tracking a vehicle's location over a specific period.

---

### **Track Link Creation**:

- **Select a vehicle** and define the **validity period** for the tracking link.
- Tap **Share** to generate the tracking link and share it with others.
- To remove the link, use the **Delete Button**.
