# Vehicle List

---

- **Steps to View the Vehicle List**:

  - Tap the **List Button** to view all vehicles.
  - Filter vehicles by status:
    - Running
    - Stopped
    - Idle
    - Inactive
    - No Data

- **Details Available in the List View**:

  - Vehicle Number
  - Today's Kilometers
  - Last Data Update
  - Current Address and Status

---

- **Actions**:
  - Call Driver
  - Immobilize Vehicle
  - View AC, Battery, and Fuel Levels
