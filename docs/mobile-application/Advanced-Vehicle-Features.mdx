# Advanced Features

---

The **Advanced Features** section provides additional tools to enhance the management of vehicles. These features offer enhanced security and tracking capabilities, enabling administrators to have more control over fleet operations.

---

### **Track Link Creation**:

This feature allows you to create a trackable link for a selected vehicle. You can set a validity period for the link, share it with others, and remove it when no longer needed.

- **Select a vehicle**: Choose the vehicle you want to create the tracking link for.
- **Set the validity period**: Define the time frame during which the link will be active.
- **Tap **Share** to generate and share the link**: Once the link is ready, tap Share to distribute it.
- **Use the **Delete Button** to remove the link**: When the tracking link is no longer needed, click the Delete button to deactivate it.

### **Immobilization**:

The Immobilization feature allows you to remotely disable a vehicle's engine to prevent it from being used. You can easily toggle this feature on or off to secure the vehicle.

- **Enable or disable the vehicle immobilization feature**: Use the toggle switch to activate or deactivate the immobilization feature when necessary.

### **Parking Mode**:

Parking Mode is a security feature that alerts you if an unauthorized movement of the vehicle occurs while parked. You can enable notifications to receive alerts in such cases.

- **Enable notifications for unauthorized vehicle movements**: Activate this feature to be notified when your vehicle is moved without authorization.
