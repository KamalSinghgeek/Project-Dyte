# Advanced Features
---
- **Track Link Creation**:
  - Select a vehicle and set the validity period.
  - Tap **Share** to generate and share the link.
  - Use the **Delete Button** to remove the link.
    <img src="/assets/guide-images/Track.png" alt="Advanced Features Image" width="250" />

---

- **Immobilization**:
  - Enable or disable the vehicle immobilization feature using the toggle.
    <img src="/assets/guide-images/Lock.png" alt="Advanced Features Image" width="250" />

---

- **Parking Mode**:
  - Enable notifications for unauthorized vehicle movements.

<img src="/assets/guide-images/Parking.png" alt="Advanced Features Image" width="250" />
