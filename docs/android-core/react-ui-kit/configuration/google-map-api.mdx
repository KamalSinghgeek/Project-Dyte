# Google Map API Setup Guide

## Overview

This guide walks you through creating and enabling a Google Maps API key with reverse geocoding functionality. 

---

## Steps to Set Up Google Maps API

### Step 1: Open Google Cloud Console

1. Navigate to [Google Cloud Console](https://cloud.console.google.com).
2. Log in using your Google Cloud Platform (GCP) account credentials.

### Step 2: Create a New Project

1. In the Google Cloud Console, click on the project drop-down menu.
2. Select **Create New Project**.
3. Add a name for your project and confirm.

### Step 3: Enable Geocoding API

1. In the Google Cloud Console, navigate to the **APIs and Services** > **Library** section.
2. Search for **Geocoding API**.
3. Select the API and click the **Enable** button.

### Step 4: Create an API Key

1. Go to the **APIs and Services** > **Credentials** section.
2. Click the **Create Credentials** button and select **API Key**.

### Step 5: Restrict the API Key

For security purposes, restrict your API key usage:

1. Under **Application Restrictions**, select **HTTP Referrers (Websites)**.
2. Add the websites or IP addresses from which you intend to make requests.
   - If using the API key in a web application, add your website URL.

### Step 6: Enable Billing

1. Go to the **Billing** section in Google Cloud Console.
2. Link a billing account to the project.

---

## Final Step: Use the API Key

You can now use the API key to access Google Maps services, including the Reverse Geocoding API. Insert the API key in your application wherever needed.

**Note:** Ensure to monitor usage to avoid exceeding your quota or incurring unexpected charges.
