# Payment Gateway Setup Guide

### 1. Access the Super Admin Panel

- **Log In**:
  - Click the **Super Admin** button on the login page.
  - Enter your Super Admin credentials.
- **Verify Software Status**:
  - Confirm whether the software is active or inactive.

### 2. Navigate to Software Configuration

- On the dashboard, locate and click the **Software Configuration** button.

### 3. Enter Payment Gateway Details

- In the **Payment Gateway** section:
  - Enter the **Razorpay Merchant ID**.
  - Enter the **Razorpay Secret Key**.

### 4. Validate Configuration

- Double-check that:
  - The **Merchant ID** is entered correctly.
  - The **Secret Key** matches the credentials provided by Razorpay.

### 5. Test the Payment Gateway (Optional but Recommended)

- Conduct a test transaction using:
  - Razorpay's test environment or sandbox mode.
  - Ensure the integration functions as expected.

### 6. Save Configuration

- After verifying the details, click the **Save** button (or equivalent) to store the settings.

### 7. Monitor and Update as Needed

- Periodically review the payment gateway settings:
  - Check for updates or changes in Razorpay credentials.
  - Adjust configurations if Razorpay requires updated API keys.

This process ensures a seamless integration of the Razorpay Payment Gateway, enabling efficient transaction handling within the Fleet Stack system.
