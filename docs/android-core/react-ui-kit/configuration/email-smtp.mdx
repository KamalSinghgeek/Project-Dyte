# Email SMTP Configuration Guide

## Introduction

Follow these steps to configure the SMTP for sending emails using your Gmail account within the Fleet Stack software.

## Steps for Email SMTP Setup

### 1. Access Your Google Account

1. Go to [Google Account Settings](https://myaccount.google.com).
2. Sign in using your Gmail account.
3. Click on **Manage your Gmail account**.

### 2. Enable Two-Step Verification

1. Navigate to the **Security** section in the left sidebar.
2. Locate **Two-Step Verification** under the signing-in options.
3. Follow the instructions to enable two-step verification.
   - Add a second layer of security using your phone number or another method to receive verification codes.

### 3. Generate App-Specific Password

1. Scroll to the **Signing into Google** section.
2. Click on **App Passwords**.
   - If this option is unavailable, ensure **Two-Step Verification** is enabled.
3. Enter your app's name and click **Create**.
4. Copy the generated 16-character app password and save it securely.
   - Note: Google will not display this password again.

### 4. Configure SMTP Settings

1. Open the Fleet Stack software and navigate to the SMTP configuration section.
2. Fill in the following details:

   - **Encryption Type**: SSL
   - **SMTP Host URL**: smtp.gmail.com
   - **SMTP Port**: 587
   - **Email**: Your Gmail email address
   - **Sender Name**: Name displayed for sent emails
   - **Username**: Your Gmail email address
   - **Password**: The app-specific password generated earlier

3. Click on **Update SMTP Credentials** to save the configuration.

### 5. Test SMTP Configuration

1. Use the **Test SMTP** section to verify the setup.
2. Send a test email to ensure the credentials work correctly.

## Usage

The configured SMTP will now enable email functionality for various features within the Fleet Stack software. Ensure the credentials remain secure and update them if your Google account settings change.
