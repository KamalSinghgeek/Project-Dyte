# Firebase API Setup Guide

## Steps to Configure Firebase API

### 1. Set Up a Firebase Project

- **Visit the Firebase Console**: Open the [Firebase Console](https://console.firebase.google.com/).
- **Add a Project**:
  - Click on **ADD PROJECT**.
  - Follow the setup wizard to create a Firebase project.
  - Provide a **project name**.
  - Uncheck the Google Analytics account option if prompted.

### 2. Configure Project Settings

- Click on the **gear icon** and navigate to **Project Settings**.
- Open a new tab and go to the **Cloud Messaging** section.
- **Enable the Cloud Messaging API Key**:
  - Click on the **Enable** button.

### 3. Generate the Key Pair

- In the **Cloud Messaging** tab, click on **Generate Key Pair**.
- Copy the **API Key** value from the **Web Push Certificate** section.

### 4. Update Configuration Files

- Navigate to the Fleet Stack web root folder:
  - Go to `C:\Program Files\Project AIQ Fleet Stack\Web Root Folder`.
- Open the following files in a text editor like **Sublime Text**:
  - Firebase Configuration File
  - Firebase Messaging File
- Paste the copied **API Key** value into the Firebase configuration file.

### 5. Register the Web Application

- Go back to the **Firebase Console**:
  - Click on the **Web Icon** to add a web app.
- **Register the Web Application**:
  - Enter the name of the web application.
  - Click on **Register**.

### 6. Add Firebase SDK

- In the Firebase Console:
  - Navigate to the **Firebase SDK** section for the web app you registered.
  - Copy the provided SDK code.
- Paste this code into the Firebase configuration and messaging files in the Fleet Stack project folder.

### 7. Enable Server Key in Fleet Stack

- Open the **Firebase Project Settings** again.
- Go to the **Cloud Messaging** section and copy the **Server Key**.
- In the Fleet Stack Super Admin panel:
  - Navigate to the **Configuration Setup** section.
  - Paste the **Server Key** into the configuration setup.

This will successfully enable the Firebase API key functionality for the Fleet Stack software.
