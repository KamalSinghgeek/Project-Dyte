# **Tracking Link Creation**:

- Select a vehicle and create Tracking Link.
- Tap **Share** to generate and share the link.
- Use the **Delete Button** to remove the link.

# Immobilize Vehicle

- **Remotely Immobilize Vehicle**:
- **Start Engine** (Green)
- **Stop Engine** (Red)
- **Note**: "It may take 2-5 minutes to fetch result from server."
